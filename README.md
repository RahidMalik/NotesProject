# NotesApp

A full-stack **Notes application** built with **React (Vite) frontend** and **Express + MongoDB backend**.  
Users can **add, edit, delete, and view notes** with live updates. Tailwind CSS is used for styling.

---

- Add new notes (title + content)  
- Edit existing notes  
- Delete notes  
- Fetch and display notes from backend  
- Responsive UI with Tailwind CSS  
- Live backend reload with Nodemon  
- Frontend hot reload via Vite
